# SCRWNLNET 電話サービス

SCRWNLNET 電話サービスはねじくぎが管理する実験用 IP
電話交換局であり、東京広域電話網に接続されています。 最新の接続状況は
[mantela.json](/.well-known/mantela.json) をご覧ください。

## 稼働環境

| 機能   | 環境                 |
| ------ | -------------------- |
| 計算機 | Raspberry Pi Zero WH |
| PBX    | Asterisk             |
